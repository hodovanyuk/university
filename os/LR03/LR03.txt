2.1 Примитивы доступа к файлам в системе UNIX

Упражнение 2.1. Создайте небольшую программу, описанную выше. Проверьте ее работу при условии, что файл junk не существует. Затем создайте файл junk с помощью текстового  редактора и снова запустиете программу. Содержимое файла junk не имеют значения.

-- Без файла в каталоге --
hodovanyuk:~/workspace/LR03 $ gcc 2_1.c

hodovanyuk:~/workspace/LR03 $ ./a.out
 
There is no file hodovanyuk:~/workspace/LR03 $ 


-- С файлом в каталоге --
Вывод: 0
